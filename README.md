# Petals — Peaceful Momentum Todo App

Welcome to **Petals**, a minimal, aesthetic, and peaceful task manager built using modern web technologies. This project is structured to offer smooth transitions, gentle visual progression (a blooming cherry blossom tree), and a clean environment to manage daily tasks.
